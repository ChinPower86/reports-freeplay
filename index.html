Как добавить новый отчёт
// Инструкция
1
Залей HTML-файл отчёта в корень репозитория reports-freeplay через GitHub (кнопка Add file)
2
Открой index.html, скопируй блок карточки и заполни дату, заголовок, теги, ссылку на файл
3
Сохрани — через ~30 секунд изменения появятся на сайте автоматически
// Именование файлов
→
Формат: trends_ДД_ММ_ГГ.html
→
Пример: trends_09_04_26.html
→
